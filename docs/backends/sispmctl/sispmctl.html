The sispmctl backend

Backend for the SIS-PM Control for Linux aka sispmct application that can control GEMBIRD (m)SiS-PM device, witch are USB controled multiple socket.

Configuration

You can load the backend by editing your config.json to include:

{ 
   "module": "sispmctl"
}

in the backend section. For all configuration options see backend-config-shema

Actuators can be added bei adding them to the actuators section in the config file. Set the class attribute to PilightOutlet. For example:

{ 
  "id": "light",
  "class": "SispmctlSwitch", 
  "name": "Lamp",
  "outletId": 1 
}

For actuator configuration options see the actuator-config-shema.coffee file.
