The filebrowser frontend

A simpe web file browser for a local directory

Example config:

{
   "module": "filebrowser",
   "mappings": [
     {
       "path": "/files",
       "directory": "/media/usb1"
     }
   ]
 }

With this config you can browse your files in /media/usb1 through http://your-ip/files
